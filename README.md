## 🎵 Introduction

The goal of this library is to abstract away obvious steps involved in the process of acting as an intermediary between a [Discord bot](https://discord.com/developers/docs/events/gateway) and [Lavalink](https://lavalink.dev/api) to provide a simple and intuitive interface to work with.

## 🛠️ Requirements

Built for [Node.js](https://nodejs.org) v22+ — [Bun](https://bun.com) v1+ and [Deno](https://deno.com) v2+ may also work. Pairs with any [Discord gateway client](https://docs.discord.com/developers/events/gateway) capable of sending and receiving raw payloads — no lock-in to a specific library.
